Quick Start
============

This guide will help you get started with pycircdb and offer a quick look at the outputs generated by pycircdb. This taster is not a comprehensive guide - refer to the rest of the documentation for more information.

Install pycircdb
----------------

1. Installing uv:

.. code-block:: bash

    # macOS / Linux
    curl -LsSf https://astral.sh/uv/install.sh | sh

    # Windows
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. Installing pycircdb

.. code-block:: bash

    uv tool install pycircdb

Verify Installation
^^^^^^^^^^^^^^^^^^^

Check that it worked by printing the pycircdb help text:

.. code-block:: bash

    pycircdb --help


Running on example data
-----------------------

To try pycircdb on example data, run the following command to create some example input data in your current working directory:

.. code-block:: bash

    pycircdb init-demo


You should now have a folder called `test/` and a file called `test_config.json` in your cwd. The `test/` folder contains example input data and the `test_config.json` file contains a configuration for running pycircdb on this data.

The command `pycircdb init-demo` will suggest a command to run, feel free to alter this command to view different outputs.

.. code-block:: bash

    pycircdb -c test_config.json -v 2 annotate -d 'arraystar,circbase' fasta -d 'arraystar,circbase' mirna -a 'miRanda,TargetScan' rbp
